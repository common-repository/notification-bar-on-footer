Show notification on the footer when scoll down the page, also hide when scroll up.

== Description ==
This plugin allow you to place notification bar on footer. It shows when you scroll down the page and disapear when scroll up. You can modify the notification bar from settings->footer Notification. You can use notification bar to show some policy or note at footer with button to redirect to some page


== Installation ==
Simply download plugin from wordpress.org or goto plugins and install and activate from there.

e.g.
1. Upload `notification-bar-on-footer.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Use of Notification bar =
You can use notification bar to show some policy or note at footer with button to redirect to some page.

= It can be use for corona update? =
Yes you can put some note there, also you can create any page for updated corona policy and use its link to this button to to read more.

= It can be use for website policy? =
Yes you can put some note there, also you can create any page for updated policy and use its link to this button to to read more to goto that page.
